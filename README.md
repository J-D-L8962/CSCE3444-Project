# CSCE3444-Project_AISmartBuddy
This project will be a mobile app that helps students organize assignments, exams, and study schedules using AI-generated study plans based on deadlines, workload, and student availability. 

The app can also provide smart reminders and productivity suggestions. The app may also include workload tracking and smart recommendations to help students reduce procrastination and manage stress more effectively. It could also help students identify busy weeks in advance and recommend better time management strategies.

This App will use databases to store student information and assignments. The functions to integrate the assignment creation function and store that data in a database, as well as the password validation functions, are written primarily in C, with the main being C++
